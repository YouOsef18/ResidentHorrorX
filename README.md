# ResidentHorrorX

Интерактивный 3D хоррор-шутер от первого лица, разработанный в рамках лабораторной работы по дисциплине ОПД (Основы профессиональной деятельности). Проект представляет собой техническую демонстрацию интеграции механик искусственного интеллекта, систем вооружения и модульного левелдизайна на движке Unreal Engine 5.

## 🛠 Технологический стек и фичи

1.  Контроль версий тяжелых данных (Git LFS): Репозиторий полностью настроен на отслеживание бинарных ассетов (меши .uasset, карты .umap, текстуры, аудиофайлы .wav) через .gitattributes. Это предотвращает раздувание истории Git и обеспечивает стабильный пулл проекта всеми участниками.
2.  Архитектура World Partition: Использование продвинутой системы разбиения мира Unreal Engine 5 позволило команде параллельно работать над одной сценой. Конфликты слияния веток локализовались на уровне отдельных файлов-акторов в директории ExternalActors, что минимизировало риски затирания чужой работы.
3.  Искусственный интеллект (Enemy AI): Настроена логика преследования игрока. Для оптимизации производительности на целевых платформах (включая Mac) внедрен режим Only Tick Pose When Rendered.
4.  Кастомная система звука: В блупринт дробовика BP_Shotgun4 интегрирован уникальный аудиоресурс выстрела, кастомизированный через звуковые параметры движка.
